# 🌸 Iris Flower Classification using Support Vector Machine (SVM)

This project implements a classification model to predict the species of iris flowers using a Support Vector Machine (SVM). It uses the famous Iris dataset and applies various data processing and visualization techniques using Python libraries.

---

## 📌 Objective

To classify iris flowers into three categories (Setosa, Versicolor, and Virginica) based on:
- Sepal length
- Sepal width
- Petal length
- Petal width

---

## 🛠️ Technologies & Libraries Used

- Python
- Pandas
- Seaborn
- Scikit-learn
- Pickle
- Matplotlib

---

## 🧪 Features

- Load and explore the Iris dataset
- Visualize data distributions using Seaborn
- Train a Support Vector Machine (SVM) classifier
- Evaluate model accuracy using confusion matrix and classification report
- Save trained model using `pickle`
